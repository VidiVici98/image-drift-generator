# Contributing

Thanks for your interest in contributing to image-drift-generator!

- Fork the repo and create a branch for your change.
- Run `make setup` then `make install` to create a venv and install deps.
- Run `make test` and `make lint` before submitting a pull request.
- Keep changes small and focused; document behavior in the README.

If you're adding features, open an issue first to discuss design.
